# Design Reference

This directory contains the original design files for the 1RM Calculator application.

## Files

### [code.html](./code.html)
Complete HTML implementation of the design with:
- Tailwind CSS styling
- Dark mode theme
- Responsive layout
- Interactive components
- Material Symbols icons

### [screen.png](./screen.png)
Screenshot of the design reference showing:
- Exercise selection interface
- Unit toggle (Lbs/Kg)
- Weight and reps input fields
- 1RM result display
- Percentage calculations (90%, 80%, 70%)

## Usage

These files serve as the design reference for implementing the actual application. The HTML file demonstrates:
- Color palette and theming
- Component structure
- Layout and spacing
- Animations and transitions
- Typography and iconography
